# Split Brain

> 멀티테스킹력을 테스트하고 기를 수 있는 간단한 모바일게임

### 개발기간
- 2025.08.21 ~ 2025.08.24 (1차 빌드)
- 3일간의 짧은 개발기간으로 최대한 기능구현에 의미를 둠

### 게임 정보
- 장르 : 캐쥬얼 아케이드 게임
- 플랫폼 : Android / iOS
- 싱글플레이

### 기술스택
- IDE : Visual Studio Code
- GameEngine : Unity 2022.3.14f1
- Language : C#
- VCS : GitHub

## 게임 시스템구조 아키텍처
- 책임의 분리 : GameManager가 전체 게임 상태와 흐름을 제어하고 실제 플레이 로직은 별도 모듈로 분리하여 관리
- 의존성 최소화 : 점수 획득(ScoreManager)와 UI 표시(UIManager) 간의 직접적인 참조를 끊기 위해 옵저버 패턴을 사용하여, 데이터가 변하면 자동으로 UI가 갱신되도록 설계
